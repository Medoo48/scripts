# Регулярні вирази зі скрипту `chg` для виправлення тексту

`/^  },/`
`/^  }/`
`/^\[/`
`/^]/`

### Прибрати : після номеру розділу
`/(Розділ [0-9]+.*):/\1./`

### Відокремити назву розділу від іншого тексту
`/(Розділ.*)/\n\1\n/g`
